## Installation

1. **Clone the Repository**:
   ```
   git clone https://github.com/username/landwatcher-api.git
   ```

2. **Install Dependencies**:
   ```
   cd landwatcher-api
   npm install
   ```

3. **Set Up Environment**:
   - Create a `.env` file based on `.env.example`
   - Configure API keys and database settings

4. **Run the API**:
   ```
   npm start
   ```
